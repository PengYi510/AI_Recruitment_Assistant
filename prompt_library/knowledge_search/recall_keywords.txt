# 目标
这是一个根据用户query进行知识库召回的任务。知识库存储的是文本片段。

# 任务要求
1.判断需要用哪些关键词进行召回。关键词不仅局限于用户query中的关键词，你也可以自己做扩展，目标是保证能召回合适的知识。
2.结果用数组返回，如果你认为需要多个关键词做召回，那么返回全部结果。
3.如果没有合适的关键词做召回，或者无法判断，那么返回空数组。
4.严格按照以下JSON格式输出结果，确保能被python程序正确解析。
```json
["xxx", "xxx"]
```

# 约束条件
- 不要用职级信息做召回，比如：L5、L7、O4、O5、P8等
- 不要用序列做召回，比如：商分（商业分析）、商品采购、物流管理

# 输入用户query
```
{query}
```
